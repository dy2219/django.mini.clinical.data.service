Home

Welcome to the minimal demo project of clinical data service.

This mini project provides the following features:

User accounts management

Django's build-in account management system is used. The administrator can add/delete/change user accounts using the /admin tool.

Adding Patients Data to the Database

Patients data are stored in the sqlite database. User needs to sign-in before adding or viewing patients data. Patients data can be added by using the /add-data tab. User input is validated upon submitting.

Viewing Patients Data

User needs to sign-in to view patients data. Patients data are displayed as a plain table in the /view-data tab. Because of limited time for implementation, there is no statistics/filtering/sorting features.
